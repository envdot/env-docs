# Grok

xAI's Grok models are designed for real-time information and direct, unfiltered responses.

## Available Models

| Model | Context | Best For |
|-------|---------|----------|
| Grok-2 | 128K tokens | Complex tasks, full capability |
| Grok-2 Mini | 128K tokens | Fast responses, lighter tasks |

## Setup

1. Get an API key from [console.x.ai](https://console.x.ai)
2. Open env Settings (⌘ + ,)
3. Add your key under **xAI**
4. Select a Grok model to start chatting

## Features

### Real-Time Knowledge

Grok has access to real-time information from X (Twitter), giving it awareness of current events and trending topics.

### Direct Responses

Grok is designed to be more direct and less filtered than other models, while still being helpful.

### Fast Inference

Grok-2 Mini offers particularly fast response times for interactive use.

## Tips

<Tip>
  Grok is great for questions about current events or topics trending on social media.
</Tip>

- Use **Grok-2** for complex tasks that need full reasoning capability
- Use **Grok-2 Mini** for quick questions and faster interactions
- Grok works well for casual conversation and direct answers

## Keyboard Shortcut

Press **⌘ + 4** to quickly switch to Grok.
